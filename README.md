# ITIL Service Management Workflow
Mise en place d'un processus de gestion des incidents et demandes (Ticketing) basé sur le référentiel ITIL avec GLPI.

## Contenu du Repository
- **Database** : Structure SQL de la base de tickets.
- **Documentation** : Procédures de support, guides utilisateur (IA, Phishing) et logigrammes.
- **Présentation** : Support de formation à l'outil GLPI.
